# AWS Backup
- central place to manage and automate backups across AWS Services
- cross region backups
- cross account backupos
- PITR support
- can create backup policies (backup plans)
- tag based
## Supported
- EC2 
- EBS
- S#
- RDS
- Aurora
- DynamoDB
- EFS
- Storage Gateway
- ...

## Target
- back ups in S3


## Vault Lock
- backups can not be deleted
